    # 快递100

    url：http://juhe.im/kuaidi/info

    拼接参数：

    参数名称参数取值参数类型
    type快递码，请参考快递100码String
    postid快递单号String
    id未知，可为空String
    valicode未知，可为空String
    temp未知，可为空String

    url 示例：http://juhe.im/kuaidi/info?type=yuantong&postid=2234014274&id=1&valicode=&temp=（圆通单号为2234014274的快递，不过快递已经过期了，所以显示不了信息，可以自行套用快递单号）

    请求方式：GET

    解析：

    • message：ok 则表示没有问题，否则携带错误信息
    • nu：运单号
    • ischeck：1表示成功，0表示错误
    • com：快递拼音
    • data：具体到达的派送点，逆序
    • time：到达时间
    • ftime：到达时间
    • context：到达地点具体信息
    • location：到达地点，但取值一直为 null
